# 구문(Statement)과 표현식(Expression)

#### 표현식(Expression) : 값 + 변수 + 연산자의 조합.
* 액션을 수행하고, 결과를 리턴함. (값이 있다.)
* 값 자체가 표현식이 되며, 변수도 동일
* 함수에 파라미터를 넘겨줄 수 있으면 표현식.
  ```java
  17
  x + 3
  x
  ```

#### 구문(Statement) : 인터프리터가 실행하는 코드 단위.
* 액션을 수행하나 아무것도 리턴하지 않음. (값이 없다.)
* 함수에 파라미터를 넘겨줄 수 없으면 표현식. (단 return을 사용하면 넘겨줄 수 있다.)
  ```java
  System.out.println("a");
  x = 2;
  System.out.println(x);
  ```
